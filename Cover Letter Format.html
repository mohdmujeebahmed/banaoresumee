Format GuideBR

Header: Your name, email, phone, LinkedIn (matches your resume).

Salutation: "Dear [Hiring Manager's Name]," or "Dear Hiring Manager," if unknown.

Paragraph 1: Role you're applying for + one strong hook.

Paragraph 2: One specific, quantified proof point.

Paragraph 3: Genuine reason for interest in the company.

Sign-off: "Sincerely," or "Best regards," followed by your name.

Mistakes

Common cover letter formatting mistakes

✕

Mismatched fonts between resume and cover letter

Keep both documents visually consistent for a polished, professional impression.

✕

Skipping the salutation entirely

Always open with a proper greeting — never start directly with the first sentence of content.

✕

Exceeding one page

A cover letter should never run longer than a single page, regardless of experience level.

Pro tips

Pro tips for cover letter formatting

Match your resume's visual style

Same font and header style across both documents looks more polished as a complete application.

Use "Dear Hiring Manager" only as a fallback

Always try to find a specific name first — a quick LinkedIn search often reveals it.

Keep paragraphs short

3-5 sentences per paragraph keeps the letter scannable and easy to read quickly.

FAQ

Cover letter format — FAQ

What is the correct length for a cover letter?

One page maximum — ideally 3-4 concise paragraphs that a hiring manager can read in under a minute.

How should I address a cover letter if I don't know the name?

Use "Dear Hiring Manager," as a fallback, but always try to find a specific name first via LinkedIn or the company site.

Should my cover letter match my resume's design?

Yes — using the same font and header style across both creates a more polished, professional application.
